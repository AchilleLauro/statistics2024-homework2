Welford Recursion

La Welford recursion è un algoritmo numerico efficiente per calcolare la media e la varianza di un insieme di dati, evitando problemi di perdita di precisione che possono insorgere con calcoli iterativi convenzionali. Questo metodo è particolarmente utile quando stai aggiornando la media e la varianza durante l'aggiunta di nuovi dati, senza dover memorizzare l'intero dataset.

Welford recursion per la media e la varianza:

Siano x₁, x₂, ..., xₙ i dati, con μₙ la media e σₙ² la varianza al passo n.

Inizializzazione:

  • μ₀ = 0 (media iniziale)
  • M₂ = 0 (un valore che ci serve per aggiornare la varianza, inizialmente 0)
  • n = 0 (contatore degli elementi)

Passo di aggiornamento (per ogni nuovo valore x):

  • Incrementa il contatore: n ← n + 1
  • Aggiorna la media: μₙ = μₙ₋₁ + (xₙ - μₙ₋₁) / n
  • Aggiorna M₂: M₂ = M₂ + (xₙ - μₙ₋₁)(xₙ - μₙ)

Calcolo finale della varianza (dopo aver inserito tutti i dati):

σ² = M₂ / n
